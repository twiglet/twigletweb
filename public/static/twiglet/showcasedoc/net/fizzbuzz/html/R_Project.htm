The FizzBuzz problem: - Print the numbers from 1 to 100 - If the number is divisible by 3 print "Fizz" instead - If the number is divisible by 5 print "Buzz" instead - If the number is divisible 3 and 5 print "FizzBuzz" instead This is a gold plated "Enterprise" ready version written by Mike Minutillo, see http://codermike.com/if-something-is-worth-doing
